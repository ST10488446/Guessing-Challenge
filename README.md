Explanation of the Features and flow of the game 

-Random Number Generation: The game will generate  a random number between 1 and 100

-User Input: The player will enter a guess through the input field and press the check button to generate the number.

-Attempt Tracking: The number of attempts is displayed and is updated after each guess.

-Feedback: After each guess ,the game provides feedback to the user telling the user if the guess was too low ,too High or just right.

-Restart Button: The player has the option to replay the game by pressing the restart button or just exiting the game.
